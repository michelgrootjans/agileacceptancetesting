To run this demo of fitnesse follow these steps:
- execute demo\build.bat
- execute fitnesse\run.bat
- open fitnesse\frontpage.url

This will open the fitnesse wiki with working tests for our application. You will find running tests under
- http://localhost:8888/WorkInProgressSuite
- http://localhost:8888/AcceptanceSuite

Requirements:
Windows + .net framework 3.5 + java JRE 6.0
